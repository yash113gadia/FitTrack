1. Dashboard: Shows daily macro progress
2. Log Food: Tap + to add food (can use manual entry)
3. Barcode: Scan any food product barcode
4. AI Photo: Take photo of food for AI recognition (requires camera permission)
5. Chatbot: Ask nutrition questions
6. Profile: View and edit personal goals

Note: AI features require network connectivity and Gemini API key.
For testing without AI, all manual features work offline.
